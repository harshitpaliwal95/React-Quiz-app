## Getting Started

- Clone the project

```bash
https://github.com/harshitpaliwal95/React-Quiz-app.git
```

- Go to the project directory

```bash
  cd React-Quiz-app
```

- Create a `.env` file at the root level of the directory

```bash
  REACT_APP_JWT_SECRET = <JWT_SECRET_KEY_OF_YOUR_CHOICE>
```

- Start the server

```bash
  npm start
```
